# 🌐 Red Empresarial con VLANs, Routing, DHCP y ACLs

## 📌 Descripción
Proyecto de simulación de una red empresarial desarrollado en Cisco Packet Tracer, implementando segmentación de red mediante VLANs, enrutamiento entre redes, asignación dinámica de direcciones IP y seguridad básica.

---

## 🧩 Tecnologías utilizadas
- VLANs
- Trunk
- Inter-VLAN Routing (Router-on-a-stick)
- DHCP
- ACLs

---

## 🏢 Escenario
Se simuló una empresa con tres departamentos:
- Administración (VLAN 10)
- Ventas (VLAN 20)
- TI (VLAN 30)

---

## ⚙️ Funcionalidades
- Segmentación de red por departamentos
- Comunicación entre VLANs
- Asignación automática de IP
- Restricción de acceso entre redes mediante ACLs

---

## 🧪 Pruebas
- Comunicación exitosa entre VLANs
- Bloqueo de tráfico entre VLAN 10 y VLAN 30
