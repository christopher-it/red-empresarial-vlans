🌐 Red Empresarial con VLANs, Routing, DHCP y ACLs
📌 Descripción
Proyecto de simulación de una red empresarial desarrollado en Cisco Packet Tracer, implementando segmentación de red mediante VLANs, enrutamiento entre redes, asignación dinámica de direcciones IP y seguridad básica mediante ACLs.

🧩 Tecnologías utilizadas

VLANs (Virtual LANs)
Trunk (enlace entre switches)
Inter-VLAN Routing (Router-on-a-stick)
DHCP (asignación dinámica de IP)
ACLs (listas de control de acceso)


🏢 Escenario
Se simuló una empresa con tres departamentos:
DepartamentoVLANAdministración10Ventas20TI30

⚙️ Funcionalidades

Segmentación de red por departamentos
Comunicación entre VLANs mediante Router-on-a-stick
Asignación automática de IP con DHCP
Restricción de acceso entre redes mediante ACLs


🧪 Pruebas realizadas

✅ Comunicación exitosa entre VLANs
✅ Bloqueo de tráfico entre VLAN 10 y VLAN 30
✅ Asignación de IP automática verificada en todos los departamentos


🚀 Cómo abrir el proyecto

Tener instalado Cisco Packet Tracer
Clonar o descargar este repositorio
Abrir el archivo .pkt con Packet Tracer
